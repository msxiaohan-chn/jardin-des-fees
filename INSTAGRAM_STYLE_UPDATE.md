# Instagram 风格轮播指示器更新

## 新增功能

已将产品轮播改成 Instagram 风格：

### 桌面端
- **点点指示器** - 显示在图片底部（半透明黑色背景）
- **左右箭头** - 悬停时显示
- **点击指示器** - 可直接跳转到指定图片

### 手机端
- **点点指示器** - 始终显示在图片底部
- **左右箭头** - 始终可见（半透明）
- **滑动操作** - 左右滑动切换图片
- **点击指示器** - 可直接跳转到指定图片

## 视觉效果

- 点点指示器有半透明黑色背景，易于识别
- 当前图片对应的点点为白色，其他为灰白色
- 箭头和指示器都有平滑的过渡动画

## 部署步骤

1. **在 GitHub 上替换文件**
   - 删除 `assets/` 文件夹中的所有文件
   - 上传本文件夹中的新 `assets/` 文件夹
   - 替换 `index.html`

2. **清除浏览器缓存**
   - 按 `Ctrl + Shift + Delete` 清除所有缓存
   - 或按 `Ctrl + Shift + R` 硬刷新

3. **测试**
   - 桌面端：悬停查看箭头，点击点点或使用箭头切换
   - 手机端：滑动或点击点点切换图片
